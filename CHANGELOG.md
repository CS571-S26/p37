# Changelog

## [0.2.0] - 2026-03-17
### Changed
- Migrated project from vanilla HTML/CSS/JS to React + Vite
- Split app into components: Header, ProgressBar, AddTaskForm, FilterBar, TaskList, TaskItem
- State managed with useState and useEffect hooks
- localStorage sync moved to useEffect

## [0.1.0] - 2026-03-17
### Added
- Initial vanilla HTML/CSS/JS implementation
- Add tasks with title, description, and priority
- Complete and delete tasks with animations
- Filter by All / Active / Done / High priority
- Progress bar
- localStorage persistence
